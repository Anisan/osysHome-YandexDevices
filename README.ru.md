# YandexDevices - Интеграция устройств Yandex IoT

![YandexDevices Icon](static/YandexDevices.png)

Интеграция с платформой Yandex IoT (Quasar) для управления умными устройствами и станциями Yandex.

## Описание

Модуль `YandexDevices` предоставляет интеграцию с платформой Yandex IoT (Quasar) для платформы osysHome. Он обеспечивает управление и мониторинг умных устройств Yandex, включая Яндекс Станции, умные колонки и IoT устройства.

## Основные возможности

- ✅ **Управление устройствами**: Управление устройствами Yandex IoT
- ✅ **Управление станциями**: Управление Яндекс Станциями
- ✅ **Управление возможностями**: Управление возможностями устройств
- ✅ **OAuth аутентификация**: Аутентификация на основе QR кода
- ✅ **Связывание свойств**: Связывание возможностей устройств со свойствами объектов
- ✅ **Связывание методов**: Связывание команд устройств с методами объектов
- ✅ **Интеграция поиска**: Поиск устройств и станций
- ✅ **Поддержка виджетов**: Виджет панели управления

## Панель администратора

Модуль предоставляет административный интерфейс для:
- Просмотра устройств Yandex
- Управления Яндекс Станциями
- Настройки возможностей устройств
- Связывания возможностей со свойствами

## Аутентификация

Модуль использует аутентификацию через QR код:
1. Перейдите на страницу аутентификации
2. Отсканируйте QR код приложением Yandex
3. Подтвердите авторизацию
4. Устройства обнаруживаются автоматически

## Конфигурация

- **Токен устройства**: Токены устройств для каждой станции
- **Платформа**: Тип платформы устройства
- **Возможности**: Конфигурация возможностей устройств

## Использование

### Добавление станции

1. Перейдите в модуль YandexDevices
2. Аутентифицируйтесь через QR код
3. Станции обнаруживаются автоматически
4. Сгенерируйте токен устройства для станции
5. Свяжите возможности со свойствами объектов

## Технические детали

- **API**: API Yandex Quasar
- **Аутентификация**: OAuth с QR кодом
- **Типы устройств**: Яндекс Станции, умные колонки, IoT устройства
- **Возможности**: вкл/выкл, громкость, яркость и т.д.

Модуль предоставляет следующие действия:

- `widget` - Виджет панели управления
